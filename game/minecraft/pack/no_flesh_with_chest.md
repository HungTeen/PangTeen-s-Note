# 简介
脆骨症（No Flesh With Chest）是一个以可拆卸器官为核心玩法的整合包。
# 特色
# 内容
### 开胸器
对生命值较低的生物可以打开其胸腔，从而取出其器官。
### 器官属性
* 健康（心脏）：决定血量上限。
* 力量（肌肉）：决定基础攻击伤害。
* 速度（肌肉）：决定基础移动速度。
* 防御（肋骨）：决定基础防御力。
* 肺活量（肺）：决定气泡的上限。
* 呼吸效率（肺）：决定呼吸气泡的恢复速度。
* 消化效率（胃）：决定每个食物恢复的饥饿值。
* 营养获取效率（肠）：决定每个食物恢复的饱和度。
* 新陈代谢效率（脾脏）：影响饱和度恢复生命的速度。
* 耐力（脊柱）：决定饱和度和饥饿值消耗的速度。
* 神经效率（脊柱）：影响挖掘效率和移动速度。
* 血液过滤效率（肾）：数值不足会中毒。
* 解毒效率（肾）：影响负面效果的持续时间。
* 幸运（阑尾）：影响开宝箱等行为的运气。
### 流派
* 大胃王：根据容器内食物的饱食度和饱和度来增加本体的伤害。【鼓励玩家做各式各样的食物】
* 大运王：通过堆叠幸运值，获得更高的开宝箱等行为的运气。
* 法王：专门打魔法伤害。
# 优点
* 多样化器官：不同生物具有属性功能各异的器官，通过不同器官的组合可以实现各式各样的玩法。
* 进化器官：玩家通过进化容器可以培养自己的器官，使其属性更强大。
# 缺点
* 普通生物的器官缺乏新意：除了海鸥等少量生物外，普通生物的器官属性大同小异，缺乏新意。【降低玩家开胸意愿】
* 流程扁平化：在玩家知道怎么获取器官后，下一步就要面对各种流派的大量器官，容易不知所措。【缺乏引导】
* 数值爆炸：游戏基本只包含前期和后期，完全没有中期。
