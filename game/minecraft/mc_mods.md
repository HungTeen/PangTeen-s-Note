# 我的世界模组
## 开发人员分配
* 程序员：纯牛马
  * 模组开发：平平无奇烂大街的角色，写mod肯定得会写代码，哪怕现在AI很强
  * 数据包开发：写写 json 或者 kubejs 脚本，为玩家扩充非官方游戏内容
  * 整合包开发：整合包常常是推广模组的最佳手段
  * 后端开发：模组常常离不开官网，页面是需要维护的
  * 服务器运维：官方服务器运行维护
* 策划：常常挨喷的角色
  * 主策划：策划具体游戏流程框架，得会讲故事
  * 内容策划：框架有了之后，如何填充有趣内容
  * 数值策划：平衡性调整
  * 交互策划：设计良好的引导、交互
  * 文案策划：命名困难症
* 美术
  * 像素画糕手：常规物品、常规方块、GUI、模组 logo 等平面像素图
  * 建模砖家：blockbench、blender 建模，包括复杂生物模型、物品模型、方块模型
  * 动画设计：模型该如何做出符合逻辑的运动姿势，太难啦
  * 特效设计：环境设计、粒子效果等等
* 音效：过于稀缺，通常由各种网站搜集，或用原版音效魔改
* 测试：测试通常由开发者自己完成，当然也可以在内测阶段抓热心玩家当小白鼠~
* 翻译：模组想在国外流行，必不可少需要多国语言，这个角色通常也有热心玩家担任
* 社区运营：协助管理官方玩家社区，包括 QQ 群、Discord 群，也可收集玩家意见反馈
